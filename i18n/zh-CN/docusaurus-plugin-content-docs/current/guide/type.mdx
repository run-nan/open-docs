# 插件类型

## 概述

ONES 现在支持「单团队」和 「多团队」两种团队管理策略，所以插件的颗粒度划分也就分为了「团队级别插件」和「组织级别插件」。

### 团队级别插件

安装在哪个团队下面就在哪个团队下生效。

<Image img={require('./images/team-zh.png')} />

### 组织级别插件

在组织内所有团队都生效的插件，只需要在【组织管理】-【插件管理】下安装一次，即可对每个团队都生效。

<Image img={require('./images/organization.png')} />

### 其他类型

细心的你可能发现在【插件管理】下除了 「业务插件」以外，还有「账号同步、登录与通知」和「托管独立服务」这两部分，那么一起来看下这两部分与正常的功能插件有什么区别吧。

#### 账号同步、登录与通知

一般是用来对接第三方账号服务的。

#### 托管独立服务

用来运行实例级别的服务。如将实例存储文件服务替换成自己选型的服务。
