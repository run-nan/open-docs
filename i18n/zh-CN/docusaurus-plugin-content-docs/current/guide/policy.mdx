# 售卖策略

## 概述

ONES 现在支持四种售卖策略：「simple」、「standard」、「full_binding」和「org_scale」。

| 名称             | 描述         |
| ----------------| ------------ |
|simple         |所有用户都已授权，不需要手动授权。     |
|standard         |按坐席售卖，可独立授权。     |
|full_binding      |跟随其他产品授权。     |
|org_scale         |绑定组织Scale授权（组织已激活用户）。     |


插件支持「simple」和「standard」。

### 「simple」插件

如果插件使用该售卖策略，所有用户都可以使用该插件功能。

<Image img={require('./images/team-zh.png')} />

### 「standard」插件

如果插件使用该售卖策略，只有在授权管理被添加授权的用户才可以使用该插件功能。
