# 流水线集成配置

## 要求

|  ONES   |
| :-----: |
| v3.6.0+ |

## 概述

<Image img={require('./images/pipeline.png')} />

我们为「配置中心」-「流水线集成配置」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:pipeline_integration:new](#onessettingspipeline_integrationnew)
- [ones:settings:pipeline_integration:default](#onessettingspipeline_integrationdefault)
- [ones:settings:pipeline:new（已废弃）](#onessettingspipelinenew已废弃)
- [ones:settings:pipeline:default（已废弃）](#onessettingspipelinedefault已废弃)

## 插槽

#### ones:settings:pipeline_integration:new

**ONES 要求：** v3.14.0+

**模块路径：** 配置中心 / 流水线集成配置 / [新模块](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:pipeline_integration:default

**ONES 要求：** v3.14.0+

**模块路径：** 配置中心 / 流水线集成配置 / [默认分组](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:pipeline:new（已废弃）

**模块路径：** 配置中心 / 流水线管理配置 / [新模块](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:pipeline:default（已废弃）

**模块路径：** 配置中心 / 流水线管理配置 / [默认分组](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-pipeline_integration-default-kkuk
    title: Pipeline settings
    moduleType: ones:settings:pipeline_integration:default
    modules:
      - id: ones-settings-pipeline_integration-default-HMrW
        title: Link Jenkins
        entry: >-
          modules/ones-settings-pipeline_integration-default-kkuk/ones-settings-pipeline_integration-default-HMrW/index.html
```

**支持的配置项：**

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
- [modules（必填）](../../../../../reference/config/plugin.yaml#subModule)
