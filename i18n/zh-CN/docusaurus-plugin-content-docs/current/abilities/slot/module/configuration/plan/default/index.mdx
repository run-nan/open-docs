# 默认分组

## 要求

|   ONES    |
| :-------: |
| v3.11.15+ |

## 概述

<Image img={require('./images/default.png')} />

我们为「配置中心」-「项目集管理配置」-「[默认分组](../../index.mdx#slot-hierarchy)」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:plan:default:new](#onessettingsplandefaultnew)
- [ones:settings:plan:default:permission](#onessettingsplandefaultpermission)

## 插槽

#### ones:settings:plan:default:new

**模块路径：** 配置中心 / 项目集管理配置 / [默认分组](../../index.mdx#slot-hierarchy) / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:plan:default:permission

**模块路径：** 配置中心 / 项目集管理配置 / [默认分组](../../index.mdx#slot-hierarchy) / 权限配置

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-plan-default-permission-VLxW
    title: Permission
    moduleType: ones:settings:plan:default:permission
    entry: modules/ones-settings-plan-default-permission-VLxW/index.html
```

**支持的配置项：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
