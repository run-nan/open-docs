# 权限

## 要求

| **ONES** |
| :------: |
| v3.6.0+  |

## 概述

<Image img={require('./images/permission.png')} />

我们为「配置中心」-「项目管理配置」-「权限」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:project:permission:new](#onessettingsprojectpermissionnew)
- [ones:settings:project:permission:config](#onessettingsprojectpermissionconfig)

## 插槽

#### ones:settings:project:permission:new

**ONES 要求：** v3.11.15+

**模块路径：** 配置中心 / 项目管理配置 / 权限 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:project:permission:config

**模块路径：** 配置中心 / 项目管理配置 / 权限 / 权限配置

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-project-permission-config-du9W
    title: Permission
    moduleType: ones:settings:project:permission:config
    entry: modules/ones-settings-project-permission-config-du9W/index.html
```

**支持的配置项：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
