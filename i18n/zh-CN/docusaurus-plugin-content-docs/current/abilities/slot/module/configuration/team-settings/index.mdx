# 团队配置

## 要求

| **ONES** |
| :------: |
| v3.6.0+  |

## 概述

<Image img={require('./images/team-settings.png')} />

我们为「配置中心」-「团队配置」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:team:new](#onessettingsteamnew)
- [ones:settings:team:default](#onessettingsteamdefault)

## 插槽

#### ones:settings:team:new

**模块路径：** 配置中心 / 团队配置 / [新模块](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:team:default

**模块路径：** 配置中心 / 团队配置 / [默认分组](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-team-default-0FBl
    title: Team info
    moduleType: ones:settings:team:default
    modules:
      - id: ones-settings-team-default-V5WX
        title: Team info
        entry: >-
          modules/ones-settings-team-default-0FBl/ones-settings-team-default-V5WX/index.html
```

**支持的配置项：**

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
- [modules（必填）](../../../../../reference/config/plugin.yaml#subModule)
