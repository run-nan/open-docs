# 项目集管理配置

## 要求

|  ONES   |
| :-----: |
| v3.6.0+ |

## 概述

<Image img={require('./images/plan.png')} />

我们为「配置中心」-「项目集管理配置」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:plan:new](#onessettingsplannew)
- [ones:settings:plan:default](#onessettingsplandefault)

## 插槽

#### ones:settings:plan:new

**模块路径：** 配置中心 / 项目集管理配置 / [新模块](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:plan:default

**模块路径：** 配置中心 / 项目集管理配置 / [默认分组](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-plan-default-2LhD
    title: Plan settings
    moduleType: ones:settings:plan:default
    modules:
      - id: ones-settings-plan-default-mLu6
        title: Permission
        entry: >-
          modules/ones-settings-plan-default-2LhD/ones-settings-plan-default-mLu6/index.html
```

**支持的配置项：**

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
- [modules（必填）](../../../../../reference/config/plugin.yaml#subModule)
