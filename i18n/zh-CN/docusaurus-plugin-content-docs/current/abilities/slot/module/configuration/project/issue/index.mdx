# 工作项

## 要求

|  ONES   |
| :-----: |
| v3.6.0+ |

## 概述

<Image img={require('./images/issue.png')} />

我们为「配置中心」-「项目管理配置」-「工作项」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:project:issue:new](#onessettingsprojectissuenew)
- [ones:settings:project:issue:type](#onessettingsprojectissuetype)
- [ones:settings:project:issue:field](#onessettingsprojectissuefield)
- [ones:settings:project:issue:status](#onessettingsprojectissuestatus)
- [ones:settings:project:issue:priority](#onessettingsprojectissuepriority)
- [ones:settings:project:issue:relate](#onessettingsprojectissuerelate)

## 插槽

#### ones:settings:project:issue:new

**ONES 要求：** v3.11.15+

**模块路径：** 配置中心 / 项目管理配置 / 工作项 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:project:issue:type

**模块路径：** 配置中心 / 项目管理配置 / 工作项 / 工作项类型

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:project:issue:field

**模块路径：** 配置中心 / 项目管理配置 / 工作项 / 工作项属性

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:project:issue:status

**模块路径：** 配置中心 / 项目管理配置 / 工作项 / 工作项状态

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:project:issue:priority

**模块路径：** 配置中心 / 项目管理配置 / 工作项 / 优先级

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:project:issue:relate

**模块路径：** 配置中心 / 项目管理配置 / 工作项 / 关联关系类型

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-project-issue-type-5PXx
    title: Issue type
    moduleType: ones:settings:project:issue:type
    entry: modules/ones-settings-project-issue-type-5PXx/index.html
```

**支持的配置项：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
