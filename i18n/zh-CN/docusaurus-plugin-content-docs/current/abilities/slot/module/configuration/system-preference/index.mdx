# 系统偏好设置

## 要求

|  ONES   |
| :-----: |
| v3.6.0+ |

## 概述

<Image img={require('./images/system-preference.png')} />

我们为「配置中心」-「系统偏好设置」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:appearance:new](#onessettingsappearancenew)
- [ones:settings:appearance:default](#onessettingsappearancedefault)

## 插槽

#### ones:settings:appearance:new

**模块路径：** 配置中心 / 系统偏好设置 / [新模块](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:appearance:default

**模块路径：** 配置中心 / 系统偏好设置 / [默认分组](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-appearance-default-TxZS
    title: System preference
    moduleType: ones:settings:appearance:default
    modules:
      - id: ones-settings-appearance-default-cq77
        title: Sidebar color
        entry: >-
          modules/ones-settings-appearance-default-TxZS/ones-settings-appearance-default-cq77/index.html
```

**支持的配置项：**

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
- [modules（必填）](../../../../../reference/config/plugin.yaml#subModule)
