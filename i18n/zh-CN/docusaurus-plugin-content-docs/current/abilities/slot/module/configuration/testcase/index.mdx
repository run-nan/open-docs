# 测试管理配置

## 要求

|  ONES   |
| :-----: |
| v3.6.0+ |

## 概述

<Image img={require('./images/testcase.png')} />

我们为「配置中心」-「测试管理配置」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:testcase:new](#onessettingstestcasenew)
- [ones:settings:testcase:base](#onessettingstestcasebase)
- [ones:settings:testcase:case](#onessettingstestcasecase)
- [ones:settings:testcase:plan](#onessettingstestcaseplan)
- [ones:settings:testcase:report](#onessettingstestcasereport)

## 插槽

#### ones:settings:testcase:new

**模块路径：** 配置中心 / 测试管理配置 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:testcase:base

**模块路径：** 配置中心 / 测试管理配置 / 基础配置

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:testcase:case

**模块路径：** 配置中心 / 测试管理配置 / 测试用例

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:testcase:plan

**模块路径：** 配置中心 / 测试管理配置 / 测试计划

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:testcase:report

**模块路径：** 配置中心 / 测试管理配置 / 测试报告

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-testcase-base-VXjH
    title: Basic configuration
    moduleType: ones:settings:testcase:base
    modules:
      - id: ones-settings-testcase-base-WXHp
        title: Link project
        entry: >-
          modules/ones-settings-testcase-base-VXjH/ones-settings-testcase-base-WXHp/index.html
```

**支持的配置项：**

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
- [modules（必填）](../../../../../reference/config/plugin.yaml#subModule)
