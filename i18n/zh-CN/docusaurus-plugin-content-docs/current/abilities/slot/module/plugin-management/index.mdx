## 概述

<Image img={require('./images/plugin-management.png')} />

我们为「插件详情」提供了一些插槽，以便于为该应用定制化页面模块。

现在系统中有两处插件管理的入口：

1. 组织管理 / 插件管理 / 选择插件类型 / 点击已安装的插件右侧「详情」按钮
2. 配置中心 / 插件管理 / 选择插件类型 / 点击已安装的插件右侧「详情」按钮

当前支持的插槽：

- [settings](#settings)

## 插槽

#### settings

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

## 使用

```yaml
modules:
  - id: settings-I1fA
    title: settings
    moduleType: settings
    entry: modules/settings-I1fA/index.html
```
