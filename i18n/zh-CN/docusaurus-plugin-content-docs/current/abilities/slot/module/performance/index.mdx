## 概述

<Image img={require('./images/performance.png')} />

我们为「效能管理」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:performance:new](#onesperformancenew)
- [ones:performance:dashboard](#onesperformancedashboard)

## 插槽

#### ones:performance:new

**模块路径：** 效能管理 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:performance:dashboard

**模块路径：** 效能管理 / 仪表盘

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

**ONES 要求：** `v3.11.15+`

## 使用

```yaml
modules:
  - id: ones-performance-dashboard-cvM8
    title: Dashboard
    moduleType: ones:performance:dashboard
    entry: modules/ones-performance-dashboard-cvM8/index.html
```

**支持的配置项：**

- [disabled](../../../../reference/config/plugin.md#disabled)
