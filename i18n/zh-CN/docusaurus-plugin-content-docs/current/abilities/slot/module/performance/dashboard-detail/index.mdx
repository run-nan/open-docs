# 仪表盘详情

## 要求

| **ONES** |
| :------: |
|   3.6+   |

## 概述

<Image img={require('./images/detail.png')} />{' '}

我们为「效能管理」-「仪表盘」-「仪表盘详情」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:performance:component:dashboard:header:action:new](#onesperformancecomponentdashboardheaderactionnew)

## 插槽

#### ones:performance:component:dashboard:header:action:new

**模块路径：** 效能管理 / 仪表盘 / 仪表盘详情 / 顶部操作模块 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

## 使用

```yaml
modules:
  - id: ones-performance-component-dashboard-header-action-new-twsu
    title: Header action
    moduleType: ones:performance:component:dashboard:header:action:new
    entry: >-
      modules/ones-performance-component-dashboard-header-action-new-twsu/index.html
```
