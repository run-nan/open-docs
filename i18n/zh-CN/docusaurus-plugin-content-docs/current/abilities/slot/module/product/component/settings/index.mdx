# 产品设置

## 要求

| **ONES** |
| :------: |
|   3.6+   |

## 概述

<Image img={require('./images/settings.png')} />

我们为「产品管理」-「产品组件」-「产品设置」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:product:settings:new](#onesproductsettingsnew)
- [ones:product:settings:config](#onesproductsettingsconfig)
- [ones:product:settings:info](#onesproductsettingsinfo)
- [ones:product:settings:component](#onesproductsettingscomponent)
- [ones:product:settings:more](#onesproductsettingsmore)

## 插槽

#### ones:product:settings:new

**模块路径：** 产品管理 / 产品组件 / 产品设置 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:product:settings:config

**模块路径：** 产品管理 / 产品组件 / 产品设置 / 产品配置

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:product:settings:info

**模块路径：** 产品管理 / 产品组件 / 产品设置 / 基础信息

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:product:settings:component

**模块路径：** 产品管理 / 产品组件 / 产品设置 / 组件设置

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:product:settings:more

**模块路径：** 产品管理 / 产品组件 / 产品设置 / 更多

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-product-settings-info-m7iL
    title: Basic info
    moduleType: ones:product:settings:info
    entry: modules/ones-product-settings-info-m7iL/index.html
```

**支持的配置项：**

- [disabled](../../../../../../reference/config/plugin.md#disabled)
