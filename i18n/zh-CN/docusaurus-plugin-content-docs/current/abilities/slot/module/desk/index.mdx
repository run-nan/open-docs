# 工单管理

## 要求

|  ONES   |
| :-----: |
| v3.6.0+ |

## 概述

<Image img={require('./images/desk.png')} />

我们为「工单管理」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:desk:new](#onesdesknew)
- [ones:desk:index](#onesdeskindex)

## 插槽

#### ones:desk:new

**模块路径：** 工单管理 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:desk:index

**模块路径：** 工单管理 / 工单概览

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-desk-index-w6Q_
    title: Ticket overview
    moduleType: ones:desk:index
    entry: modules/ones-desk-index-w6Q_/index.html
```

**支持的配置项：**

- [disabled](../../../../reference/config/plugin.yaml#disabled)
