# 项目集管理

import Image from '@theme/IdealImage'

## 要求

| **ONES** |
| :------: |
| v3.6.0+  |

## 概述

<Image img={require('./images/plan.png')} />

我们为「项目集管理」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:plan:new](#onesplannew)
- [ones:plan:self](#onesplanself)
- [ones:plan:gantt](#onesplangantt)

## 插槽

#### ones:plan:new

**模块路径：** 项目集管理 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:plan:self

**ONES 要求：** v3.11.15+

**模块路径：** 项目集管理 / 项目集

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:plan:gantt

**模块路径：** 项目集管理 / 甘特图

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-plan-self-07F2
    title: Programs
    moduleType: ones:plan:self
    entry: modules/ones-plan-self-07F2/index.html
```

**支持的配置项：**

- [disabled](../../../../reference/config/plugin.md#disabled)
