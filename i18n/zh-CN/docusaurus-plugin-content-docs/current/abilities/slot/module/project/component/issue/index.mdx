# 工作项详情

## 要求

|  ONES   |
| :-----: |
| v3.6.0+ |

## 概述

<Image img={require('./images/activity.png')} />

我们为「项目管理」-「项目组件」-「工作项详情」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:issue:feed:all](#onesissuefeedall)
- [ones:issue:feed:comment](#onesissuefeedcomment)
- [ones:issue:feed:file](#onesissuefeedfile)
- [ones:issue:feed:relate](#onesissuefeedrelate)
- [ones:issue:feed:changelog](#onesissuefeedchangelog)
- [ones:issue:header:action:new](#onesissueheaderactionnew)

## 插槽

#### ones:issue:header:action:new

**ONES 要求：** v3.XX.XX+

**模块路径：** 项目管理 / 项目组件 / 工作项详情 / 顶部操作模块 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:issue:feed:all

**模块路径：** 项目管理 / 项目组件 / 工作项详情 / 动态 / 全部

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:issue:feed:comment

**模块路径：** 项目管理 / 项目组件 / 工作项详情 / 动态 / 评论

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:issue:feed:file

**模块路径：** 项目管理 / 项目组件 / 工作项详情 / 动态 / 文件

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:issue:feed:relate

**模块路径：** 项目管理 / 项目组件 / 工作项详情 / 动态 / 关联

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:issue:feed:changelog

**模块路径：** 项目管理 / 项目组件 / 工作项详情 / 动态 / 变更记录

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-issue-feed-comment-GAy9
    title: Comment
    moduleType: ones:issue:feed:comment
    entry: modules/ones-issue-feed-comment-GAy9/index.html
```

**支持的配置项：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
