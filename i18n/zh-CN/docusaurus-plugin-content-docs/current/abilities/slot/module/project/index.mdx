# 项目管理

## 要求

|  ONES   |
| :-----: |
| v3.6.0+ |

## 概述

<Image img={require('./images/projects.png')} />

我们为「项目管理」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:project:new](#onesprojectnew)
- [ones:project:index](#onesprojectindex)

## 插槽

#### ones:project:new

**模块路径：** 项目管理 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:project:index

**模块路径：** 项目管理 / 项目

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-project-index-rvh-
    title: Projects
    moduleType: ones:project:index
    entry: modules/ones-project-index-rvh-/index.html
```

**支持的配置项：**

- [disabled](../../../../reference/config/plugin.yaml#disabled)
