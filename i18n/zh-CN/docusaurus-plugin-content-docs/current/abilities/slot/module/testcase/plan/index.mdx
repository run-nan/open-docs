## 概述

<Image img={require('./images/test-case-plan.png')} />

我们为「测试管理」-「测试计划」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持插槽如下：

- [ones:testcase:plan:header:action:new](#onestestcaseplancomponentcase)

## 插槽

#### ones:testcase:plan:header:action:new

**模块路径：** 测试管理 / 测试计划

**个数限制：** 一个插件中最多可声明 **10** 个该插槽, 系统中最多同时存在 **100** 个该插槽.

## 使用

```yaml
modules:
  - id: ones:testcase:plan:header:action:new-2efs
    title: Test case
    moduleType: ones:testcase:plan:header:action:new
    entry: modules/ones:testcase:plan:header:action:new-2efs/index.html
```
