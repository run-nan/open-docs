# 知识库管理

## 要求

|  ONES   |
| :-----: |
| v3.6.0+ |

## 概述

<Image img={require('./images/wiki.png')} />

我们为「知识库管理」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:wiki:new](#oneswikinew)
- [ones:wiki:group](#oneswikigroup)
- [ones:wiki:share](#oneswikishare)

## 插槽

#### ones:wiki:new

**模块路径：** 知识库管理 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:wiki:group

**ONES 要求：** v3.11.15+

**模块路径：** 知识库管理 / 页面组

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:wiki:share

**模块路径：** 知识库管理 / 与我共享

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-wiki-group-YjhC
    title: Spaces
    moduleType: ones:wiki:group
    entry: modules/ones-wiki-group-YjhC/index.html
```

**支持的配置项：**

- [disabled](../../../../reference/config/plugin.yaml#disabled)
