## 插槽

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:app:new

**模块路径：** 系统左侧菜单栏

**个数限制：** 一个插件中最多可声明 **1 **个该插槽

:::tip

- 该插槽的名称支持通过 `plugin.yaml` 中的 modules 的 **title** 属性进行配置
- 该插槽需配合能力一起使用，具体请参见[侧边栏选项](../../basic/app-menu.mdx)
- 如果你想要替换整个右侧区域，包括顶部区域，`plugin.yaml` 需要配置如下：

  ```yaml
  modules:
    - id: ones-workspace-9rMc
      title: My work
      moduleType: ones:app:new
      entry: modules/ones-workspace-9rMc/index.html
  ```

  这个功能从 `ONES v3.16.8+` 开始支持。

:::

## 使用

```yaml
modules:
  - id: ones-workspace-9rMc
    title: My work
    moduleType: ones:workspace
    modules:
      - id: ones-workspace-A0r3
        title: Overview
        entry: modules/ones-workspace-9rMc/ones-workspace-A0r3/index.html
```

**支持的配置项：**

- [modules（必填）](../../../reference/config/plugin.yaml#subModule)

:::caution 注意

`title`的配置项不能重命名系统侧边栏菜单的名称，需要在「配置中心」-「系统偏好设置」-「侧边栏菜单」中配置。

:::
