# 全屏展示

## 要求

| ONES      |
| :-------- |
| v3.13.35+ |

## 概述

在某些场景下为了聚焦信息的展示，你可能需要隐藏左侧菜单栏和顶部菜单栏，突出中间页面的信息。我们给你提供了非常简单的方法来实现全屏的功能。

<Image img={require('./images/full-screen.png')} />

## 使用

使用它的方法很简单，例如，要访问测试计划页面，你需要在地址栏中输入以下地址：

```shell
https://ones.com/project/#/testcase/team/${teamUUID}/plan
```

当你想要以全屏形式访问测试计划页面时，只需键入：

```shell
https://ones.com/project/#/testcase/team/${teamUUID}/plan?allow=fullscreen
```

你只需在正常访问地址后添加查询参数 **`allow=fullscreen`** 即可。

:::tip
如果你想退出全屏模式，只需清除地址栏上的 **`allow=fullscreen`** 查询参数，刷新页面即可。
:::
