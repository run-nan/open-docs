# 获取插件能力配置

## 要求

| **ONES** |
| :------- |
| 待发布   |

## 概述

插件可以通过该能力获取插件的能力配置信息

能力配置信息位于插件配置文件`plugin.yaml`中

## 使用

### 第一步：安装依赖

进入插件工程的`/backend`目录，执行以下命令进行依赖安装：

```shell
npm i @ones-op/node-ability
```

### 第二步：获取能力配置信息

通过`getAbilityConfig`方法获取能力配置信息

```typescript
import { getAbilityConfig } from '@ones-op/node-ability'

const response = await getAbilityConfig()
```

该能力允许通过指定某一个能力 id 来获取指定能力配置信息

:::info

能力 id 可以在插件配置文件`plugin.yaml`中获取

:::

```typescript
import { getAbilityConfig } from '@ones-op/node-ability'

const response = await getAbilityConfig(abilityID)
```
