# Auth



ONES REST API采用标准的OAuth 2.0授权机制，旨在为第三方应用程序提供安全的用户授权和访问令牌，以访问受保护的资源。OAuth 2.0允许第三方应用程序以一种安全、可控的方式获取有限且可撤销的访问权限，而无需直接使用用户的凭证。这种授权机制提供了一种灵活且可靠的方式，使得第三方应用程序可以与ONES系统进行安全互动，并获取所需的资源访问权限。
