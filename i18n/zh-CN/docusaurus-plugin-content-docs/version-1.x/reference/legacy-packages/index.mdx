# 注意事项

一些注意事项。

## 目录说明

此目录归档了所有废弃能力包文档，仅为旧插件保留相关能力文档说明，不再更新。

当你的新插件在满足新能力包最低使用要求的情况下，你不应该使用此目录下的能力包。

## 语义化

我们提供的能力包遵循 [Semantic Versioning 2.0.0](http://semver.org/lang/zh-CN/) 语义化版本规范。

在能力包达到 1.0 版本之前，破坏性更改将以新的次要版本发布。 例如 0.5.1 和 0.5.4 将具有相同的 API，但 0.6.0 将具有重大变化。
