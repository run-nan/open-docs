# 系统顶栏右侧插槽

## 要求

|   ONES   |
| :------: |
| v3.6.11+ |

## 概述

<Image img={require('./images/header.png')} />{' '}

我们在系统顶栏右侧提供了一些插槽，以便于定制化系统顶栏右侧模块

当前支持的插槽：

- [ones:layout:header:action:new](#oneslayoutheaderactionnew)
- [ones:layout:header:action:search](#oneslayoutheaderactionsearch)（已废弃）
- [ones:layout:header:action:add](#oneslayoutheaderactionadd)
- [ones:layout:header:action:help](#oneslayoutheaderactionhelp)（已废弃）
- [ones:layout:header:action:settings](#oneslayoutheaderactionsettings)（已废弃）
- [ones:layout:header:action:notification](#oneslayoutheaderactionnotification)（已废弃）
- [ones:layout:header:action:avatar](#oneslayoutheaderactionavatar)（已废弃）

:::danger 已废弃
从`v3.14.0+`开始，该功能已被弃用。

- ones:layout:header:action:search
- ones:layout:header:action:help
- ones:layout:header:action:settings
- ones:layout:header:action:notification
- ones:layout:header:action:avatar

:::

## 插槽

#### ones:layout:header:action:new

**模块路径：** 系统顶栏右侧操作模块，用于在系统顶栏右侧追加新操作内容

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:layout:header:action:search（已废弃）

**模块路径：** 系统顶栏右侧操作模块，用于替换系统顶栏右侧「搜索」操作模块

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:layout:header:action:add

**模块路径：** 系统顶栏右侧操作模块，用于替换系统顶栏右侧「新建」操作模块

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:layout:header:action:help（已废弃）

**模块路径：** 系统顶栏右侧操作模块，用于替换系统顶栏右侧「帮助」操作模块

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:layout:header:action:settings（已废弃）

**模块路径：** 系统顶栏右侧操作模块，用于替换系统顶栏右侧「设置」操作模块

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:layout:header:action:notification（已废弃）

**模块路径：** 系统顶栏右侧操作模块，用于替换系统顶栏右侧「通知」操作模块

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:layout:header:action:avatar（已废弃）

**模块路径：** 系统顶栏右侧操作模块，用于替换系统顶栏右侧「头像」操作模块

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-layout-header-action-new-nYKZ
    title: action
    moduleType: ones:layout:header:action:new
    entry: modules/ones-layout-header-action-new-nYKZ/index.html
```
