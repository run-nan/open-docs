# 筛选器

## 要求

| **ONES**  |
| :-------: |
| v3.10.26+ |

## 概述

<Image img={require('./images/filter.png')} />

在「我的工作台」-「筛选器」组件的顶部操作模块的左侧追加新内容

当前支持的插槽：

- [ones:workspace:filter:header:action:new](#onesworkspacefilterheaderactionnew)

## 插槽

#### ones:workspace:filter:header:action:new

**模块路径：** 我的工作台 / 筛选器 / 顶部操作模块 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

## 使用

```yaml
modules:
  - id: ones-workspace-filter-header-action-new-Nc4T
    title: new header action
    moduleType: ones:workspace:filter:header:action:new
    entry: modules/ones-workspace-filter-header-action-new-Nc4T/index.html
```
