# 产品组件

## 要求

| **ONES** |
| :------: |
| v3.6.0+  |

## 概述

<Image img={require('./images/component.png')} />

我们为「产品管理」-「产品组件」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:product:component:version](#onesproductcomponentversion)
- [ones:product:component:desk](#onesproductcomponentdesk)
- [ones:product:component:story](#onesproductcomponentstory)
- [ones:product:component:bug](#onesproductcomponentbug)
- [ones:product:component:task](#onesproductcomponenttask)
- [ones:product:component:demand](#onesproductcomponentdemand)
- [ones:product:component:module](#onesproductcomponentmodule)

## 插槽

#### ones:product:component:version

**模块路径：** 产品管理 / 产品组件 / 版本

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:product:component:desk

**模块路径：** 产品管理 / 产品组件 / 工单

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:product:component:story

**模块路径：** 产品管理 / 产品组件 / 用户故事

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:product:component:bug

**模块路径：** 产品管理 / 产品组件 / 缺陷

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:product:component:task

**模块路径：** 产品管理 / 产品组件 / 任务

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:product:component:demand

**模块路径：** 产品管理 / 产品组件 / 需求

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:product:component:module

**模块路径：** 产品管理 / 产品组件 / 模块

## 使用

```yaml
modules:
  - id: ones-product-component-story-Aa2M
    title: User story
    moduleType: ones:product:component:story
    entry: modules/ones-product-component-story-Aa2M/index.html
```
