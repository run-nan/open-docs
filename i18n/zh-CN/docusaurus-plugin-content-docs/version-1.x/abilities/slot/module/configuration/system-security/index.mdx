# 系统安全

## 要求

|  ONES   |
| :-----: |
| v3.6.0+ |

## 概述

<Image img={require('./images/system-security.png')} />

我们为「配置中心」-「系统安全」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:safe:new](#onessettingssafenew)
- [ones:settings:safe:default](#onessettingssafedefault)

## 插槽

#### ones:settings:safe:new

**模块路径：** 配置中心 / 系统安全 / [新模块](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:safe:default

**模块路径：** 配置中心 / 系统安全 / [默认分组](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-safe-default-48rW
    title: System security
    moduleType: ones:settings:safe:default
    modules:
      - id: ones-settings-safe-default-CaHY
        title: Audit Log
        entry: >-
          modules/ones-settings-safe-default-48rW/ones-settings-safe-default-CaHY/index.html
```

**支持的配置项：**

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
- [modules（必填）](../../../../../reference/config/plugin.yaml#subModule)
