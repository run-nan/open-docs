# 代码集成配置

## 要求

|   ONES   |
| :------: |
| v3.14.0+ |

## 概述

<Image img={require('./images/code-integration.png')} />

我们为「配置中心」-「代码集成配置」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:code_integration:new](#onessettingscode_integrationnew)
- [ones:settings:code_integration:default](#onessettingscode_integrationdefault)

## 插槽

#### ones:settings:code_integration:new

**模块路径：** 配置中心 / 代码集成配置 / [新模块](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:code_integration:default

**模块路径：** 配置中心 / 代码集成配置 / [默认分组](../index.mdx#slot-hierarchy)

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-code_integration-default-kkuk
    title: Code Integration settings
    moduleType: ones:settings:code_integration:default
    modules:
      - id: ones-settings-code_integration-default-HMrW
        title: Link repositories
        entry: >-
          modules/ones-settings-code_integration-default-kkuk/ones-settings-code_integration-default-HMrW/index.html
```

**支持的配置项：**

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
- [modules（必填）](../../../../../reference/config/plugin.yaml#subModule)
