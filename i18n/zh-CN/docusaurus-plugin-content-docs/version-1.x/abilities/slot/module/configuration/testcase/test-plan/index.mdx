# 测试计划

## 要求

| **ONES** |
| :------: |
| v3.6.0+  |

## 概述

<Image img={require('./images/test-plan.png')} />

我们为「配置中心」-「测试管理配置」-「测试计划」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:testcase:plan:new](#onessettingstestcaseplannew)
- [ones:settings:testcase:plan:field](#onessettingstestcaseplanfield)

## 插槽

#### ones:settings:testcase:plan:new

**ONES 要求：** v3.11.15+

**模块路径：** 配置中心 / 测试管理配置 / 测试计划 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:testcase:plan:field

**模块路径：** 配置中心 / 测试管理配置 / 测试计划 / 测试计划属性

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-testcase-plan-field-o_zn
    title: Test plan field
    moduleType: ones:settings:testcase:plan:field
    entry: modules/ones-settings-testcase-plan-field-o_zn/index.html
```

**支持的配置项：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
