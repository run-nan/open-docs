# 基础配置

## 要求

| **ONES** |
| :------: |
| v3.6.0+  |

## 概述

<Image img={require('./images/basic.png')} />

我们为「配置中心」-「测试管理配置」-「基础配置」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:testcase:base:new](#onessettingstestcasebasenew)
- [ones:settings:testcase:base:project](#onessettingstestcasebaseproject)
- [ones:settings:testcase:base:permission](#onessettingstestcasebasepermission)

## 插槽

#### ones:settings:testcase:base:new

**ONES 要求：** v3.11.15+

**模块路径：** 配置中心 / 测试管理配置 / 基础配置 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:testcase:base:project

**模块路径：** 配置中心 / 测试管理配置 / 基础配置 / 关联项目配置

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:testcase:base:permission

**模块路径：** 配置中心 / 测试管理配置 / 基础配置 / 权限配置

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-testcase-base-project-ugP9
    title: Link project
    moduleType: ones:settings:testcase:base:project
    entry: modules/ones-settings-testcase-base-project-ugP9/index.html
```

**支持的配置项：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
