## 概述

<Image img={require('./images/organization-settings.png')} />

我们为「组织管理」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:organization:new](#onessettingsorganizationnew)
- [ones:settings:organization:manage](#onessettingsorganizationmanage)
- [ones:settings:organization:account](#onessettingsorganizationaccount)
- [ones:settings:organization:appearance](#onessettingsorganizationappearance)
- [ones:settings:organization:safe](#onessettingsorganizationsafe)

## 插槽

#### ones:settings:organization:new

**模块路径：** 组织管理 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:organization:manage

**模块路径：** 组织管理 / 组织管理

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:organization:account

**模块路径：** 组织管理 / 账号和成员

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:organization:appearance

**模块路径：** 组织管理 / 系统偏好设置

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:organization:safe

**模块路径：** 组织管理 / 系统安全

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-organization-manage-y4dV
    title: Organization settings
    moduleType: ones:settings:organization:manage
    modules:
      - id: ones-settings-organization-manage-R7q8
        title: Organization info
        entry: >-
          modules/ones-settings-organization-manage-y4dV/ones-settings-organization-manage-R7q8/index.html
```

**支持的配置项：**

- [disabled](../../../../reference/config/plugin.yaml#disabled)
- [modules（必填）](../../../../reference/config/plugin.yaml#subModule)
