# 账号和成员

## 要求

| **ONES** |
| :------: |
| v3.6.0+  |

## 概述

<Image img={require('./images/accounts-and-members.png')} />

我们为「组织管理」-「账号和成员」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:settings:organization:account:new](#onessettingsorganizationaccountnew)
- [ones:settings:organization:account:manage](#onessettingsorganizationaccountmanage)
- [ones:settings:organization:account:login](#onessettingsorganizationaccountlogin)
- [ones:settings:organization:account:connect](#onessettingsorganizationaccountconnect)

## 插槽

#### ones:settings:organization:account:new

**模块路径：** 组织管理 / 账号和成员 / 新模块

**个数限制：** 一个插件中最多可声明 **10 **个该插槽，系统中最多同时存在 **100 **个该插槽

#### ones:settings:organization:account:manage

**模块路径：** 组织管理 / 账号和成员 / 组织成员管理

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:organization:account:login

**模块路径：** 组织管理 / 账号和成员 / 登录登出设置

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:settings:organization:account:connect

**模块路径：** 组织管理 / 账号和成员 / 账号集成

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-settings-organization-account-manage-AYTo
    title: Manage organization members
    moduleType: ones:settings:organization:account:manage
    entry: modules/ones-settings-organization-account-manage-AYTo/index.html
```

**支持的配置项：**

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
