我们为「登录」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

- [ones:login](#oneslogin)
- [ones:join:team](#onesjointeam)
- [ones:login:ldap](#onesloginldap)
- [ones:login:connect](#onesloginconnect)

## 插槽

#### ones:login

**ONES 要求：** v3.6.25+

**模块路径：** 替换整个登录页面

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:join:team

**ONES 要求：** v3.6.25+

**模块路径：** 替换整个“加入团队”的登录页面

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:login:ldap

**ONES 要求：** v3.15.7

**模块路径：** 替换整个 Ldap 登录页面

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

#### ones:login:connect

**模块路径：** 新增系统登录页的第三方登录入口

**个数限制：** 一个插件中最多可声明 **1 **个该插槽，系统中最多同时存在 **1 **个该插槽

## 使用

```yaml
modules:
  - id: ones-login-connect-RCp-
    title: login
    moduleType: ones:login:connect
    entry: modules/ones-login-connect-RCp-/index.html
```
