概览

ONES CLI 和 OP CLI 是一个命令行界面工具，可帮助开发者构建 ONES 开放平台插件。

ONES CLI 可以快速初始化 ONES 开放平台插件工程，在插件工程初始化的过程 OP CLI 会作为插件工程的项目依赖被安装，开发者可以使用它来执行许多常见的开发任务。

开发者需要使用 Node.js 安装 ONES CLI 和 OP CLI，要了解如何使用 ONES CLI 与 OP CLI 执行其他任务，请参阅以下文档：
