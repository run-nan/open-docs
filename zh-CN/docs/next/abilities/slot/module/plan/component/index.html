项目集组件

要求​

ONES
v3.6.0+

概述​

我们为「项目集管理」-「项目集」-「项目集组件」提供了一些插槽，以便于为该应用定制化页面模块。

当前支持的插槽：

  • ones:plan:component:project
  • ones:plan:component:member

插槽​

ones:plan:component:project​

模块路径： 项目集管理 / 项目集 / 项目集组件 / 项目

个数限制： 一个插件中最多可声明 1 个该插槽，系统中最多同时存在 1 个该插槽

ones:plan:component:member​

模块路径： 项目集管理 / 项目集 / 项目集组件 / 成员

个数限制： 一个插件中最多可声明 1 个该插槽，系统中最多同时存在 1 个该插槽

使用​

modules:
  - id: ones-plan-component-project-tblX
    title: Project
    moduleType: ones:plan:component:project
    entry: modules/ones-plan-component-project-tblX/index.html

可访问的上下文数据​

  • useProgramInfo
import { useProgramInfo } from '@ones-op/store'
