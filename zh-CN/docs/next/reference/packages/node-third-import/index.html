版本：🚧 Canary

@ones-op/node-third-import

要求​

ONES
v3.11.39+

安装​

进入插件工程的/backend目录，执行以下命令进行依赖安装：

npm i @ones-op/node-third-import

API​

startImport​

开始三方数据导入

Params​

参数类型说明必填默认值
userUUIDstring用户 uuid是-
abilityIdstring插件能力配置中的能力 id是-
passwordstring首次登录时导入用户的默认密码（默认密码需要是字母数字组合）是-
serverIdstring如果导入的是同一批数据需要保证有相同的 server_id是-
teamUUIDstring团队唯一标识，仅为组织插件时需要否-

Example​

import { startImport } from '@ones-op/node-third-import'

await startImport(user_uuid, abilityId, password, server_id, teamUUID)
