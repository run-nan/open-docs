平台开放能力

概览​

📄️ 进度管理器

支持使用进度管理器来处理队列任务或异步任务。

🗃️ 托管独立应用

2 个项目

📄️ 插件日志

允许每个插件打印自己的日志，提供了 4 种级别的标准日志打印方法： debug、info、warning、error，并且支持导出日志文件。

📄️ 数据库

有时候插件希望可以持久化自己的数据。插件数据库能力可以让每个插件拥有自己独立的数据库，并提供访问和操作它的方法。

📄️ 发送外部通知

插件具备向用户发送消息通知的能力，支持的通知渠道有微信、钉钉、有度、飞书和邮件。

📄️ 插件自定义配置

插件可以使用该能力在“配置中心 -> 插件管理”下自定义添加插件的配置项。

📄️ 获取插件能力配置

插件可以通过该能力获取插件的能力配置信息。

📄️ 超级用户

每个插件在 ONES 中都有一个专属的超级用户，该用户拥有 ONES 所有数据的访问权限。当插件在使用某些能力时，需要使用较高权限的 ONES 用户，可以直接使用插件超级用户而不需要预先配置特定的用户。

📄️ 定时器

提供简便的定时任务使用方法。

📄️ 审计日志

插件可以向 ONES 写入审计日志，在“配置中心 -> 系统安全 -> 操作日志”页面，可以看到审计日志内容。插件使用该能力产生的审计日志的操作类型为插件日志。

📄️ 上传文件

用户可以通过该能力上传文件（允许大于 10M）到插件存储空间 workspace 下。

📄️ 下载文件

从插件工作空间下载文件。

📄️ 插件文件操作
