Testcase Library

此资源代表测试用例库。使用它来获取、创建、更新和删除测试用例库。

Authentication​

  • OAuth 2.0: oauth2
Security Scheme Type:oauth2
OAuth Flow (authorizationCode):

Token URL: https://your-domain/oauth2/token

Authorization URL: https://your-domain/oauth2/authorize

Scopes:
  • write:testcase:library: 新增、修改、删除用例库
  • read:testcase:library: 获取用例库信息
