获取工作项附件列表

Path Parameters
    teamID string required

    团队ID

    issueID string required

    工作项ID

Query Parameters
    fileCreatedTimeSince integer

    文件创建时间最小值

    fileCreatedTimeMax integer

    文件创建时间最大值

    limit integer

    本次请求要返回的数据数量

    cursor string

    本次请求的起始位置

Responses
  • 200
  • 401
  • 403
  • 404
  • 500

请求成功
