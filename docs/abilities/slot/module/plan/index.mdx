| **ONES** |
| :------: |
| v3.6.0+  |

## Overview

<Image img={require('./images/plan.png')} />

We have provided slots for "ONES Plan" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:plan:new](#onesplannew)
- [ones:plan:self](#onesplanself)
- [ones:plan:gantt](#onesplangantt)

## Slots

#### ones:plan:new

**Module path：** ONES Plan / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:plan:self

**ONES Requirement:** v3.11.15+

**Module path：** ONES Plan / Programs

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:plan:gantt

**Module path：** ONES Plan / Gantt

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-plan-self-07F2
    title: Programs
    moduleType: ones:plan:self
    entry: modules/ones-plan-self-07F2/index.html
```

**Supported config options：**

- [disabled](../../../../reference/config/plugin.yaml#disabled)
