# Pipeline component

## Requirements

| **ONES** |
| :------: |
| v3.6.0+  |

## Overview

<Image img={require('./images/component.png')} />

We have provided slots for "ONES Pipeline" - "Pipeline" - "Pipeline component" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:pipeline:component:history](#onespipelinecomponenthistory)

## Slots

#### ones:pipeline:component:history

**Module path：** ONES Pipeline / Pipeline / Pipeline component / Running history

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-pipeline-component-history-1IXS
    title: Running history
    moduleType: ones:pipeline:component:history
    entry: modules/ones-pipeline-component-history-1IXS/index.html
```

### Available Hook API

- [usePipelineInfo](../../../../../reference/packages/store/store.md#usePipelineInfo)

```ts
import { usePipelineInfo } from '@ones-op/store'

const pipelineInfo = usePipelineInfo()
```
