# ONES Pipeline

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/pipeline.png')} />

We have provided slots for "ONES Pipeline" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:pipeline:new](#onespipelinenew)
- [ones:pipeline:index](#onespipelineindex)

## Slots

#### ones:pipeline:new

**Module path：** ONES Pipeline / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:pipeline:index

**Module path：** ONES Pipeline / Pipeline

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-pipeline-index-CF4J
    title: Pipeline
    moduleType: ones:pipeline:index
    entry: modules/ones-pipeline-index-CF4J/index.html
```

**Supported config options：**

- [disabled](../../../../reference/config/plugin.yaml#disabled)
