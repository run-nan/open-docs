# Dashboards

## Requirements

| **ONES** |
| :------: |
|   3.6+   |

## Overview

<Image img={require('./images/dashboard.png')} />{' '}

Append a new module to the left of the top header action zone under the path "My work" - "Dashboards"

The slots currently supported are as follows:

- [ones:workspace:dashboard:header:action:new](#onesworkspacedashboardheaderactionnew)

## Slots

#### ones:workspace:dashboard:header:action:new

**Module path：** My work / Dashboards / top header action / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

## Usage

```yaml
modules:
  - id: ones-workspace-dashboard-header-action-new-nW_y
    title: new action header
    moduleType: ones:workspace:dashboard:header:action:new
    entry: modules/ones-workspace-dashboard-header-action-new-nW_y/index.html
```

### Available Hook API

- [useDashboardInfo](../../../../../reference/packages/store/store.md#useDashboardInfo)

import { useDashboardInfo } from '@ones-op/store'

const dashboardInfo = useDashboardInfo()
