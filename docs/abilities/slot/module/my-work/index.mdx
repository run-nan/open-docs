# My work

## Requirements

| **ONES** |
| :------: |
| v3.6.0+  |

## Overview

<Image img={require('./images/my-work.png')} />

We have provided slots for "My work" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:workspace:new](#onesworkspacenew)
- [ones:workspace:index](#onesworkspaceindex)
- [ones:workspace:dashboard](#onesworkspacedashboard)
- [ones:workspace:filter](#onesworkspacefilter)
- [ones:workspace:manhour](#onesworkspacemanhour)

## Slots

#### ones:workspace:new

**Module path：** My work / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:workspace:overview

**Module path：** My work / Overview

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:workspace:dashboard

**Module path：** My work / Dashboards

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:workspace:filter

**Module path：** My work / Filters

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:workspace:manhour

**Module path：** My work / Worklog

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-workspace-overview-NP56
    title: Overview
    moduleType: ones:workspace:overview
    entry: modules/ones-workspace-overview-NP56/index.html
```

**Supported config options：**

- [disabled](../../../../reference/config/plugin.yaml#disabled)
