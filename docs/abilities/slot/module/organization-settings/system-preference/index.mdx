# System preference settings

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/system-preference.png')} />

We have provided slots for "Organization settings" - "System preference settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:organization:appearance:new](#onessettingsorganizationappearancenew)
- [ones:settings:organization:appearance:sidebar](#onessettingsorganizationappearancesidebar)
- [ones:settings:organization:appearance:favicon](#onessettingsorganizationappearancefavicon)
- [ones:settings:organization:appearance:email](#onessettingsorganizationappearanceemail)
- [ones:settings:organization:appearance:message](#onessettingsorganizationappearancemessage)

## Slots

#### ones:settings:organization:appearance:new

**Module path：** Organization settings / System preference settings / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:organization:appearance:sidebar

**Module path：** Organization settings / System preference settings / Sidebar color

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:organization:appearance:favicon

**Module path：** Organization settings / System preference settings / Browser favicon

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:organization:appearance:email

**Module path：** Organization settings / System preference settings / Email settings

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:organization:appearance:message

**Module path：** Organization settings / System preference settings / Mobile number and SMS settings

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-organization-appearance-sidebar-L67m
    title: Sidebar color
    moduleType: ones:settings:organization:appearance:sidebar
    entry: modules/ones-settings-organization-appearance-sidebar-L67m/index.html
```

**Supported config options：**

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
