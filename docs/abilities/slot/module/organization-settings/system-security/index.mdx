# System security

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/system-security.png')} />

We have provided slots for "Organization settings" - "System security" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:organization:safe:new](#onessettingsorganizationsafenew)
- [ones:settings:organization:safe:log](#onessettingsorganizationsafelog)
- [ones:settings:organization:safe:policy](#onessettingsorganizationsafepolicy)
- [ones:settings:organization:safe:watermark](#onessettingsorganizationsafewatermark)

## Slots

#### ones:settings:organization:safe:new

**Module path：** Organization settings / System security / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:organization:safe:log

**Module path：** Organization settings / System security / Audit Log

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:organization:safe:policy

**Module path：** Organization settings / System security / Network access security policy

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:organization:safe:watermark

**Module path：** Organization settings / System security / Watermark

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-organization-safe-log-1KjM
    title: Audit Log
    moduleType: ones:settings:organization:safe:log
    entry: modules/ones-settings-organization-safe-log-1KjM/index.html
```

**Supported config options：**

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
