We have provided slots for "Login" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:login](#oneslogin)
- [ones:login:ldap](#onesloginldap)
- [ones:login:connect](#onesloginconnect)

## Slots

#### ones:login

**ONES Requirement:** v3.6.25+

**Module path：** Replace the login page.

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:login:ldap

**ONES Requirement:** v3.15.7

**Module path：** Replace the ldap login page.

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:login:connect

**Module path：** Add a new entrance in the third-party login module.

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-login-connect-RCp-
    title: login
    moduleType: ones:login:connect
    entry: modules/ones-login-connect-RCp-/index.html
```
