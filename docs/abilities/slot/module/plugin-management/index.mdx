# Plugin management

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/plugin-management.png')} />

We have provided slots for "plugin detail" to customize the page module for the application.

There are two entries for "Plugin management" in ONES.

1. Organization settings / Plugin management / select the plugin's type / click the "detail" button on the right of the installed plugin
2. Configuration / Plugin management / select the plugin's type / click the "detail" button on the right of the installed plugin

The slots currently supported are as follows:

- [settings](#settings)

## Slots
