## Overview

<Image img={require('./images/plan.png')} />

We have provided slots for "ONES TestCase" - "Test plans" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:testcase:plan:component:case](#onestestcaseplancomponentcase)
- [ones:testcase:plan:component:demand](#onestestcaseplancomponentdemand)

## Slots

#### ones:testcase:plan:component:case

**Module path：** ONES TestCase / Test plans / Test case

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:testcase:plan:component:demand

**Module path：** ONES TestCase / Test plans / Requirements traceability

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-testcase-plan-component-case-2efs
    title: Test case
    moduleType: ones:testcase:plan:component:case
    entry: modules/ones-testcase-plan-component-case-2efs/index.html
```

**Supported config options：**

- [disabled](../../../../../reference/config/plugin.md#disabled)
