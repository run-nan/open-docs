## Overview

<Image img={require('./images/settings.png')} />

We have provided slots for "ONES TestCase" - "Test plans" - "Test plan settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:testcase:plan:settings:new](#onestestcaseplansettingsnew)
- [ones:testcase:plan:settings:info](#onestestcaseplansettingsinfo)
- [ones:testcase:plan:settings:perform](#onestestcaseplansettingsperform)
- [ones:testcase:plan:settings:permission](#onestestcaseplansettingspermission)
- [ones:testcase:plan:settings:more](#onestestcaseplansettingsmore)

## Slots

#### ones:testcase:plan:settings:new

**Module path：** ONES TestCase / Test plans / Test plan settings / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:testcase:plan:settings:info

**Module path：** ONES TestCase / Test plans / Test plan settings / Test plan info

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:testcase:plan:settings:perform

**Module path：** ONES TestCase / Test plans / Test plan settings / Execution configuration

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:testcase:plan:settings:permission

**Module path：** ONES TestCase / Test plans / Test plan settings / Permission

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:testcase:plan:settings:more

**Module path：** ONES TestCase / Test plans / Test plan settings / More

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-testcase-plan-settings-info-c6mm
    title: Info
    moduleType: ones:testcase:plan:settings:info
    entry: modules/ones-testcase-plan-settings-info-c6mm/index.html
```

**Supported config options：**

- [disabled](../../../../../../reference/config/plugin.md#disabled)
