# Personal info

## Requirements

| **ONES** |
| :------: |
| v3.6.0+  |

## Overview

<Image img={require('./images/personal.png')} />

We have provided slots for "My profile" - "Personal info" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:account:personal:new](#onesaccountpersonalnew)
- [ones:account:personal:profile](#onesaccountpersonalprofile)
- [ones:account:personal:connect](#onesaccountpersonalconnect)

## Slots

#### ones:account:personal:new

**Module path：** My profile / Personal info / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:account:personal:profile

**Module path：** My profile / Personal info / My account

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:account:personal:connect

**Module path：** My profile / Personal info / Third party account

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-account-personal-profile-c6kB
    title: My account
    moduleType: ones:account:personal:profile
    entry: modules/ones-account-personal-profile-c6kB/index.html
```
