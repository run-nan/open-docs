# Plan

## Requirements

|   ONES    |
| :-------: |
| v3.13.46+ |

## Overview

<Image img={require('./images/header-action.png')} />

We support appending a new module to the left of the top header action zone under the path "ONES Project" - "Project component" - "Project plan".

The slots currently supported are as follows:

- [ones:project:component:plan:header:action:new](#onesprojectcomponentplanheaderactionnew)

## Slots

#### ones:project:component:plan:header:action:new

**Module path: ** ONES Project / Project component / Project plan / top header action / new module

**Number limit: ** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

## Usage

```yaml
modules:
  - id: ones-project-component-plan-action-new-r6pG
    title: new header action
    moduleType: ones:project:component:plan:header:action:new
    entry: modules/ones-project-component-plan-action-new-r6pG/index.html
```
