## Overview

<Image img={require('./images/header-action.png')} />

We support appending a new module to the left of the top header action zone under the path "ONES Project" - "Project component" - "Overview".

The slots currently supported are as follows:

- [ones:project:component:overview:header:action:new](#onesprojectcomponentoverviewheaderactionnew)

## Slots

#### ones:project:component:overview:header:action:new

**Module path：** ONES Project / Project component / Overview / top header action / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

## Usage

```yaml
modules:
  - id: ones-project-component-overview-header-action-new-NavM
    title: new header action
    moduleType: ones:project:component:overview:header:action:new
    entry: modules/ones-project-component-overview-header-action-new-NavM/index.html
```
