# ONES Project

## Requirements

| **ONES** |
| :------: |
|   3.6+   |

## Overview

<Image img={require('./images/projects.png')} />

We have provided slots for "ONES Project" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:project:new](#onesprojectnew)
- [ones:project:index](#onesprojectindex)

## Slots

#### ones:project:new

**Module path：** ONES Project / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:project:index

**Module path：** ONES Project / Projects

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-project-index-rvh-
    title: Projects
    moduleType: ones:project:index
    entry: modules/ones-project-index-rvh-/index.html
```

**Supported config options：**

- [disabled](../../../../reference/config/plugin.md#disabled)
