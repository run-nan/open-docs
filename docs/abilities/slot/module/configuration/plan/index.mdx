# Plan settings

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/plan.png')} />

We have provided slots for "Configuration" - "Plan settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:plan:new](#onessettingsplannew)
- [ones:settings:plan:default](#onessettingsplandefault)

## Slots

#### ones:settings:plan:new

**Module path：** Configuration / Plan settings / [new module](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:plan:default

**Module path：** Configuration / Plan settings / [Default group](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-plan-default-2LhD
    title: Plan settings
    moduleType: ones:settings:plan:default
    modules:
      - id: ones-settings-plan-default-mLu6
        title: Permission
        entry: >-
          modules/ones-settings-plan-default-2LhD/ones-settings-plan-default-mLu6/index.html
```
