# Code Integration settings

## Requirements

|   ONES   |
| :------: |
| v3.14.0+ |

## Overview

<Image img={require('./images/code-integration.png')} />

We have provided slots for "Configuration" - "Code Integration settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:code_integration:new](#onessettingscode_integrationnew)
- [ones:settings:code_integration:default](#onessettingscode_integrationdefault)

## Slots

#### ones:settings:code_integration:new

**Module path：** Configuration / Code Integration settings / [new module](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:code_integration:default

**Module path：** Configuration / Code Integration settings / [Default group](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-code_integration-default-kkuk
    title: Code Integration settings
    moduleType: ones:settings:code_integration:default
    modules:
      - id: ones-settings-code_integration-default-HMrW
        title: Link repositories
        entry: >-
          modules/ones-settings-code_integration-default-kkuk/ones-settings-code_integration-default-HMrW/index.html
```
