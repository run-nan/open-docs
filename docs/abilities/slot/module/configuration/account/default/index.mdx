# Default group

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/default.png')} />

We have provided slots for "Configuration" - "User management" - "[Default group](../../index.mdx#slot-hierarchy)" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:account:default:new](#onessettingsaccountdefaultnew)
- [ones:settings:account:default:member](#onessettingsaccountdefaultmember)
- [ones:settings:account:default:group](#onessettingsaccountdefaultgroup)
- [ones:settings:account:default:login](#onessettingsaccountdefaultlogin)
- [ones:settings:account:default:connect](#onessettingsaccountdefaultconnect)
- [ones:settings:account:default:safe](#onessettingsaccountdefaultsafe)

## Slots

#### ones:settings:account:default:new

**Module path：** Configuration / User management / [Default group](../../index.mdx#slot-hierarchy) / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:account:default:member

**Module path：** Configuration / User management / [Default group](../../index.mdx#slot-hierarchy) / Member

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:account:default:group

**Module path：** Configuration / User management / [Default group](../../index.mdx#slot-hierarchy) / User group

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:account:default:login

**Module path：** Configuration / User management / [Default group](../../index.mdx#slot-hierarchy) / Login abd logout settings

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:account:default:connect

**Module path：** Configuration / User management / [Default group](../../index.mdx#slot-hierarchy) / Account integrations

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:account:default:safe

**Module path：** Configuration / User management / [Default group](../../index.mdx#slot-hierarchy) / Account security

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-account-default-member-6QCI
    title: Member
    moduleType: ones:settings:account:default:member
    entry: modules/ones-settings-account-default-member-6QCI/index.html
```
