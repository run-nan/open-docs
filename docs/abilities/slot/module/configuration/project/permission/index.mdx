# Permission

## Requirements

| **ONES** |
| :------: |
| v3.6.0+  |

## Overview

<Image img={require('./images/permission.png')} />

We have provided slots for "Configuration" - "Project settings" - "Permission" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:project:permission:new](#onessettingsprojectpermissionnew)
- [ones:settings:project:permission:config](#onessettingsprojectpermissionconfig)

## Slots

#### ones:settings:project:permission:new

**ONES Requirement:** v3.11.15+

**Module path：** Configuration / Project settings / Permission / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:project:permission:config

**Module path：** Configuration / Project settings / Permission / Permission

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-project-permission-config-du9W
    title: Permission
    moduleType: ones:settings:project:permission:config
    entry: modules/ones-settings-project-permission-config-du9W/index.html
```

**Supported config options：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
