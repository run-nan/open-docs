# Project

## Requirements

| **ONES** |
| :------: |
| v3.6.0+  |

## Overview

<Image img={require('./images/project.png')} />

We have provided slots for "Configuration" - "Project settings" - "Project" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:project:self:new](#onessettingsprojectselfnew)
- [ones:settings:project:self:manage](#onessettingsprojectselfmanage)
- [ones:settings:project:self:field](#onessettingsprojectselffield)
- [ones:settings:project:self:status](#onessettingsprojectselfstatus)
- [ones:settings:project:self:role](#onessettingsprojectselfrole)

## Slots

#### ones:settings:project:self:new

**ONES Requirement:** v3.11.15+

**Module path：** Configuration / Project settings / Project / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:project:self:manage

**Module path：** Configuration / Project settings / Project / Project management

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:project:self:field

**Module path：** Configuration / Project settings / Project / Project field

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:project:self:status

**Module path：** Configuration / Project settings / Project / Project status

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:project:self:role

**Module path：** Configuration / Project settings / Project / Project role

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-project-self-manage-KEU_
    title: Project management
    moduleType: ones:settings:project:self:manage
    entry: modules/ones-settings-project-self-manage-KEU_/index.html
```
