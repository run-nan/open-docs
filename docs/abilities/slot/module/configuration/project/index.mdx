# Project settings

## Requirements

| **ONES** |
| :------: |
| v3.6.0+  |

## Overview

<Image img={require('./images/project.png')} />

We have provided slots for "Configuration" - "Project settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:project:new](#onessettingsprojectnew)
- [ones:settings:project:self​](#onessettingsprojectself)
- [ones:settings:project:issue](#onessettingsprojectissue)
- [ones:settings:project:product](#onessettingsprojectproduct)
- [ones:settings:project:config](#onessettingsprojectconfig)
- [ones:settings:project:permission](#onessettingsprojectpermission)

## Slots

#### ones:settings:project:new

**Module path：** Configuration / Project settings / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:project:self​

**Module path：** Configuration / Project settings / Project

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:project:issue

**Module path：** Configuration / Project settings / Issue

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:project:product

**Module path：** Configuration / Project settings / Product

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:project:config

**Module path：** Configuration / Project settings / Global configuration

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:project:permission

**Module path：** Configuration / Project settings / Permission

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-project-self-KnvX
    title: Project
    moduleType: ones:settings:project:self
    modules:
      - id: ones-settings-project-self-MxpO
        title: Project management
        entry: >-
          modules/ones-settings-project-self-KnvX/ones-settings-project-self-MxpO/index.html
```
