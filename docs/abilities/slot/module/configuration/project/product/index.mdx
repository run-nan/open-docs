# Product

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/product.png')} />

We have provided slots for "Configuration" - "Project settings" - "Product" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:project:product:new](#onessettingsprojectproductnew)
- [ones:settings:project:product:field](#onessettingsprojectproductfield)

## Slots

#### ones:settings:project:product:new

**ONES Requirement:** v3.11.15+

**Module path：** Configuration / Project settings / Product / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:project:product:field

**Module path：** Configuration / Project settings / Product / Product field

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-project-product-field-y02J
    title: Product field
    moduleType: ones:settings:project:product:field
    entry: modules/ones-settings-project-product-field-y02J/index.html
```

**Supported config options：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
