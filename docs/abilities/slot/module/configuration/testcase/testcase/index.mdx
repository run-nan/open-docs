# Test case

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/testcase.png')} />

We have provided slots for "Configuration" - "TestCase settings" - "Test case" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:testcase:case:new](#onessettingstestcasecasenew)
- [ones:settings:testcase:case:field](#onessettingstestcasecasefield)
- [ones:settings:testcase:case:config](#onessettingstestcasecaseconfig)

## Slots

#### ones:settings:testcase:case:new

**ONES Requirement:** v3.11.15+

**Module path：** Configuration / TestCase settings / Test case / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:testcase:case:field

**Module path：** Configuration / TestCase settings / Test case / Test case field

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:testcase:case:config

**Module path：** Configuration / TestCase settings / Test case / Test case field configuration

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-testcase-case-field-pw8b
    title: Test case field
    moduleType: ones:settings:testcase:case:field
    entry: modules/ones-settings-testcase-case-field-pw8b/index.html
```

**Supported config options：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
