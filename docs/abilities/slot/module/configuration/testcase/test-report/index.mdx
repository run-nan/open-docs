# Test report

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/test-report.png')} />

We have provided slots for "Configuration" - "TestCase settings" - "Test report" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:testcase:report:new](#onessettingstestcasereportnew)
- [ones:settings:testcase:report:template](#onessettingstestcasereporttemplate)

## Slots

#### ones:settings:testcase:report:new

**ONES Requirement:** v3.11.15+

**Module path：** Configuration / TestCase settings / Test report / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:testcase:report:template

**Module path：** Configuration / TestCase settings / Test report / Manage report template

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-testcase-report-template-gVeg
    title: Manage report template
    moduleType: ones:settings:testcase:report:template
    entry: modules/ones-settings-testcase-report-template-gVeg/index.html
```

**Supported config options：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
