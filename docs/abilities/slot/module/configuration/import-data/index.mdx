# Import data

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/import-data.png')} />

We have provided slots for "Configuration" - "Import data" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:import:new](#onessettingsimportnew)
- [ones:settings:import:default](#onessettingsimportdefault)

## Slots

#### ones:settings:import:new

**Module path：** Configuration / Import data / [new module](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:import:default

**Module path：** Configuration / Import data / [Default group](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-import-default-QMQ6
    title: Import data
    moduleType: ones:settings:import:default
    modules:
      - id: ones-settings-import-default-hn01
        title: Import data from Jira
        entry: >-
          modules/ones-settings-import-default-QMQ6/ones-settings-import-default-hn01/index.html
```
