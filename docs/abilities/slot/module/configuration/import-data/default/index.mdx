# Default group

## Requirements

| **ONES** |
| :------: |
| v3.6.0+  |

## Overview

<Image img={require('./images/default.png')} />

We have provided slots for "Configuration" - "Import data" - "[Default group](../../index.mdx#slot-hierarchy)" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:import:default:new](#onessettingsimportdefaultnew)
- [ones:settings:import:default:jira](#onessettingsimportdefaultjira)

## Slots

#### ones:settings:import:default:new

**Module path：** Configuration / Import data / [Default group](../../index.mdx#slot-hierarchy) / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:import:default:jira

**Module path：** Configuration / Import data / [Default group](../../index.mdx#slot-hierarchy) / Import data from Jira

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-import-default-jira-O18b
    title: Import data from Jira
    moduleType: ones:settings:import:default:jira
    entry: modules/ones-settings-import-default-jira-O18b/index.html
```

**Supported config options：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
