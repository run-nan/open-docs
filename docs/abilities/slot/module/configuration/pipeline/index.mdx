# Pipeline Integration settings

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/pipeline.png')} />

We have provided slots for "Configuration" - "Pipeline Integration settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:pipeline_integration:new](#onessettingspipeline_integrationnew)
- [ones:settings:pipeline_integration:default](#onessettingspipeline_integrationdefault)
- [ones:settings:pipeline:new (deprecated)](#onessettingspipelinenew-deprecated)
- [ones:settings:pipeline:default (deprecated)](#onessettingspipelinedefault-deprecated)

## Slots

#### ones:settings:pipeline_integration:new

**ONES Requirement:** v3.14.0+

**Module path：** Configuration / Pipeline Integration settings / [new module](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:pipeline_integration:default

**ONES Requirement:** v3.14.0+

**Module path：** Configuration / Pipeline Integration settings / [Default group](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:pipeline:new (deprecated)

**Module path：** Configuration / Pipeline settings / [new module](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:pipeline:default (deprecated)

**Module path：** Configuration / Pipeline settings / [Default group](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-pipeline_integration-default-kkuk
    title: Pipeline settings
    moduleType: ones:settings:pipeline_integration:default
    modules:
      - id: ones-settings-pipeline_integration-default-HMrW
        title: Link Jenkins
        entry: >-
          modules/ones-settings-pipeline_integration-default-kkuk/ones-settings-pipeline_integration-default-HMrW/index.html
```
