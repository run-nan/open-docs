# Pipeline settings

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/pipeline.png')} />

We have provided slots for "Configuration" - "Pipeline settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:pipeline:new](#onessettingspipelinenew)
- [ones:settings:pipeline:default](#onessettingspipelinedefault)

## Slots

#### ones:settings:pipeline:new

**Module path：** Configuration / Pipeline settings / [new module](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:pipeline:default

**Module path：** Configuration / Pipeline settings / [Default group](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-pipeline-default-kkuk
    title: Pipeline settings
    moduleType: ones:settings:pipeline:default
    modules:
      - id: ones-settings-pipeline-default-HMrW
        title: Link Jenkins
        entry: >-
          modules/ones-settings-pipeline-default-kkuk/ones-settings-pipeline-default-HMrW/index.html
```
