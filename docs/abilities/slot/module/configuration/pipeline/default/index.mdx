# Default group

## Requirements

|   ONES    |
| :-------: |
| v3.11.15+ |

## Overview

<Image img={require('./images/default.png')} />

We have provided slots for "Configuration" - "Pipeline Integration settings" - "[Default group](../../index.mdx#slot-hierarchy)" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:pipeline_integration:default:new](#onessettingspipeline_integrationdefaultnew)
- [ones:settings:pipeline_integration:default:jenkins](#onessettingspipeline_integrationdefaultjenkins)
- [ones:settings:pipeline:default:new (deprecated)](#onessettingspipelinedefaultnew-deprecated)
- [ones:settings:pipeline:default:jenkins (deprecated)](#onessettingspipelinedefaultjenkins-deprecated)

## Slots

#### ones:settings:pipeline_integration:default:new

**ONES Requirement:** v3.14.0+

**Module path：** Configuration / Pipeline Integration settings / [Default group](../../index.mdx#slot-hierarchy) / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:pipeline_integration:default:jenkins

**ONES Requirement:** v3.14.0+

**Module path：** Configuration / Pipeline Integration settings / [Default group](../../index.mdx#slot-hierarchy) / Link pipeline tools

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:pipeline:default:new (deprecated)

**Module path：** Configuration / Pipeline settings / [Default group](../../index.mdx#slot-hierarchy) / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:pipeline:default:jenkins (deprecated)

**Module path：** Configuration / Pipeline settings / [Default group](../../index.mdx#slot-hierarchy) / Link Jenkins

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-pipeline_integration-default-jenkins-YA0a
    title: Link Jenkins
    moduleType: ones:settings:pipeline_integration:default:jenkins
    entry: modules/ones-settings-pipeline_integration-default-jenkins-YA0a/index.html
```

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
