# Performance settings

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/performance.png')} />

We have provided slots for "Configuration" - "Performance settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:performance:new](#onessettingsperformancenew)
- [ones:settings:performance:default](#onessettingsperformancedefault)

## Slots

#### ones:settings:performance:new

**Module path：** Configuration / Performance settings / [new module](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:performance:default

**Module path：** Configuration / Performance settings / [Default group](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-performance-default-xAF6
    title: Performance settings
    moduleType: ones:settings:performance:default
    modules:
      - id: ones-settings-performance-default-FeZK
        title: Permission
        entry: >-
          modules/ones-settings-performance-default-xAF6/ones-settings-performance-default-FeZK/index.html
```
