# Default group

## Requirements

| **ONES**  |
| :-------: |
| v3.11.15+ |

## Overview

<Image img={require('./images/default.png')} />

We have provided slots for "Configuration" - "Worklog settings" - "[Default group](../../index.mdx#slot-hierarchy)" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:manhour:default:new](#onessettingsmanhourdefaultnew)
- [ones:settings:manhour:default:mode](#onessettingsmanhourdefaultmode)
- [ones:settings:manhour:default:limit](#onessettingsmanhourdefaultlimit)

## Slots

#### ones:settings:manhour:default:new

**Module path：** Configuration / Worklog settings / [Default group](../../index.mdx#slot-hierarchy) / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:manhour:default:mode

**Module path：** Configuration / Worklog settings / [Default group](../../index.mdx#slot-hierarchy) / Worklog mode

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:manhour:default:limit

**Module path：** Configuration / Worklog settings / [Default group](../../index.mdx#slot-hierarchy) / Time spent limit

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-manhour-default-mode-h778
    title: Worklog mode
    moduleType: ones:settings:manhour:default:mode
    entry: modules/ones-settings-manhour-default-mode-h778/index.html
```

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
