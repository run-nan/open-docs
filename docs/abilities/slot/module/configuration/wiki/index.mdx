# Wiki settings

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/wiki.png')} />

We have provided slots for "Configuration" - "Wiki settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:wiki:new](#onessettingswikinew)
- [ones:settings:wiki:default](#onessettingswikidefault)

## Slots

#### ones:settings:wiki:new

**Module path：** Configuration / Wiki settings / [new module](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:wiki:default

**Module path：** Configuration / Wiki settings / [Default group](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-wiki-default-u5bl
    title: Wiki settings
    moduleType: ones:settings:wiki:default
    modules:
      - id: ones-settings-wiki-default-ktWv
        title: Space management
        entry: >-
          modules/ones-settings-wiki-default-u5bl/ones-settings-wiki-default-ktWv/index.html
```
