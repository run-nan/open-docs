# App slot

## Requirements

| **ONES** |
| :------: |
|   3.6+   |

## Overview

<Image img={require('./images/my-work.png')} />

App slot is a set of slots corresponding to product lines one by one.

The slots currently supported are as follows:

- [ones:workspace](#onesworkspace)
- [ones:project](#onesproject)
- [ones:wiki](#oneswiki)
- [ones:testcase](#onestestcase)
- [ones:product](#onesproduct)
- [ones:desk](#onesdesk)
- [ones:deskhome](#onesdeskhome)
- [ones:manhour](#onesmanhour)
- [ones:performance](#onesperformance)
- [ones:plan](#onesplan)
- [ones:pipeline](#onespipeline)
- [ones:version](#onesversion)

## Slots

#### ones:workspace

**Module path：** My work

**Number limit：** Declare a maximum of 1 slots in each plugin, and up to 1 slots in the system.

#### ones:project

**Module path：** ONES Project

**Number limit：** Declare a maximum of 1 slots in each plugin, and up to 1 slots in the system.

#### ones:desk

**Module path：** ONES DeskHome

**Number limit：** Declare a maximum of 1 slots in each plugin, and up to 1 slots in the system.

```
${location.protocol}://${location.host}/project/#/desk_home
```

## Usage

```yaml
modules:
  - id: ones-workspace-9rMc
    title: My work
    moduleType: ones:workspace
    modules:
      - id: ones-workspace-A0r3
        title: Overview
        entry: modules/ones-workspace-9rMc/ones-workspace-A0r3/index.html
```
