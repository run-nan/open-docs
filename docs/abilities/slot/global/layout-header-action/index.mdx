# Header action slot

## Requirements

|   ONES   |
| :------: |
| v3.6.11+ |

## Overview

<Image img={require('./images/header.png')} />{' '}

We have provided slots at the right top of the system in the header action to customize the module.

The slots currently supported are as follows:

- [ones:layout:header:action:new](#oneslayoutheaderactionnew)
- [ones:layout:header:action:search](#oneslayoutheaderactionsearch)
- [ones:layout:header:action:add](#oneslayoutheaderactionadd)
- [ones:layout:header:action:help](#oneslayoutheaderactionhelp)
- [ones:layout:header:action:settings](#oneslayoutheaderactionsettings)
- [ones:layout:header:action:notification](#oneslayoutheaderactionnotification)
- [ones:layout:header:action:avatar](#oneslayoutheaderactionavatar)

## Slots

#### ones:layout:header:action:new

**Module path：** Append a new slot in the header action.

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:layout:header:action:search

**Module path：** Replace the "Search" module in the header action.

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:layout:header:action:add

**Module path：** Replace the "Create" module in the header action.

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:layout:header:action:help

**Module path：** Replace the "Help" module in the header action.

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:layout:header:action:settings

**Module path：** Replace the "Settings" module in the header action.

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:layout:header:action:notification

**Module path：** Replace the "Unread notifications" module in the header action.

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:layout:header:action:avatar

**Module path：** Replace the "Avatar" module in the header action.

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-layout-header-action-new-nYKZ
    title: action
    moduleType: ones:layout:header:action:new
    entry: modules/ones-layout-header-action-new-nYKZ/index.html
```
