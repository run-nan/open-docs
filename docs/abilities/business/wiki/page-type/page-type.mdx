# Page node type

## Requirements

|   ONES    |
| :-------: |
| v3.16.18+ |

## Overview

Supports connecting new page types to WIKI through plug-ins, such as connecting to GrapeCity's table editor.

### Implementation

The standard product side wiki-api has been abstracted `editor` support dynamic addition and use third editor.

Use independent applications to implement user-defined `editor` specifications, and use plug-in support to access new page types to WIKI through the open platform.
