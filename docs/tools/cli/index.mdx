# Overview

ONES CLI and OP CLI are command-line interface tools that help developers build plugins for the ONES Open Platform.

ONES CLI can efficiently initialize the plugin project. During the initialization process of the plugin project, OP CLI will be installed as a project dependency of the plugin project. Developers can use it to perform many common development tasks.
