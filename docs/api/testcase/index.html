API 说明​

特殊说明：​

使用 GraphQL 查询，可以自行构建请求参数和筛选条件等等

GraphQL 使用方法请参考 GraphQL 文档

1. 查询用例库​

URL​

https://your-host-name/project/api/project/team/:teamUUID/items/graphql

HTTP Method​

POST

调用权限​

manage_library_cases

传值方式​

JSON

参数列表​

参数名值类型值说明是否可以空
querystringGraphQL 请求脚本否

GraphQL 请求脚本​

{
    testcaseLibraries{
        key
        uuid
        name
        isPin
        namePinyin
        createTime
        members{
            type
            param
        }
    }
}

返回值​

{
  "data": {
    "testcaseLibraries": [
      {
        "createTime": 1547783373,
        "isPin": false,

            "param": "Gq8ZZZ7F",
            "type": "single_user"
          },

            "param": "",
            "type": "testcase_administrators"
