Version: 1.x

@ones-op/node-file(deprecated)

We provide a set of API related to file manipulation, allowing plugin developers to manipulate files under the plugin file working directory.

danger

There have some bugs in the @ones-op/node-file package, please replace it with the @ones-op/node-fs package when the ONES version exceeds v3.14.0.

Requirementsâ€‹

ONES
v3.6.0+
