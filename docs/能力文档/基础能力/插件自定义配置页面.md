| ConfigurationPage | 插件自定义配置界面 |

## 能力描述

根据配置文件内容，为插件提供配置页面



## 能力使用

### 声明

在plugin.yaml中添加

```yaml
config:
	- key: select_key
  	value: ''
  	type: 102
  	required: true
```

key：配置页面的显示名称，识别该配置的唯一名称

value：配置的默认值

type：在页面显示的的配置项类型，可以是 101:单行为本，102：多行文本，103：密码文本等。

required：是否必填




### 实现

安装插件后，点击插件详情，可以在插件功能配置中添加自定义配置。
