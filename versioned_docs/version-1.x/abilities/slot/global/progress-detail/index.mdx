# Progress slot

## Requirements

|   ONES    |
| :-------: |
| v3.13.41+ |

## Overview

<Image img={require('./images/progress.png')} />{' '}

We have provided slots for the task created by the plugin in the progress bar to replace the content in the detail modal.

The slots currently supported are as follows:

- [ones:global:progress:detail](#onesglobalprogressdetail)

## Slots

#### ones:global:progress:detail

**Number limitï¼š** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

:::caution

This slot need the ability "[Progress Manager](../../../basic/progress-manager/progress-manager.mdx)"

:::

## Usage

```yaml
modules:
  - id: ones-global-progress-detail-BUTc
    title: progress
    moduleType: ones:global:progress:detail
    entry: modules/ones-global-progress-detail-BUTc/index.html
```
