# TestCase settings

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/testcase.png')} />

We have provided slots for "Configuration" - "TestCase settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:testcase:new](#onessettingstestcasenew)
- [ones:settings:testcase:base](#onessettingstestcasebase)
- [ones:settings:testcase:case](#onessettingstestcasecase)
- [ones:settings:testcase:plan](#onessettingstestcaseplan)
- [ones:settings:testcase:report](#onessettingstestcasereport)

## Slots

#### ones:settings:testcase:new

**Module path：** Configuration / TestCase settings / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:testcase:base

**Module path：** Configuration / TestCase settings / Basic configuration

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:testcase:case

**Module path：** Configuration / TestCase settings / Test case

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:testcase:plan

**Module path：** Configuration / TestCase settings / Test plan

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:testcase:report

**Module path：** Configuration / TestCase settings / Test report

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-testcase-base-VXjH
    title: Basic configuration
    moduleType: ones:settings:testcase:base
    modules:
      - id: ones-settings-testcase-base-WXHp
        title: Link project
        entry: >-
          modules/ones-settings-testcase-base-VXjH/ones-settings-testcase-base-WXHp/index.html
```
