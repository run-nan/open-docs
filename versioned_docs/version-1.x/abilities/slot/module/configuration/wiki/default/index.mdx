# Default group

## Requirements

| **ONES**  |
| :-------: |
| v3.11.15+ |

## Overview

<Image img={require('./images/default.png')} />

We have provided slots for "Configuration" - "Wiki settings" - "[Default group](../../index.mdx#slot-hierarchy)" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:wiki:default:new](#onessettingswikidefaultnew)
- [ones:settings:wiki:default:group](#onessettingswikidefaultgroup)
- [ones:settings:wiki:default:permission](#onessettingswikidefaultpermission)
- [ones:settings:wiki:default:template](#onessettingswikidefaulttemplate)

## Slots

#### ones:settings:wiki:default:new

**Module path：** Configuration / Wiki settings / [Default group](../../index.mdx#slot-hierarchy) / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:wiki:default:group

**Module path：** Configuration / Wiki settings / [Default group](../../index.mdx#slot-hierarchy) / Space management

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:wiki:default:permission

**Module path：** Configuration / Wiki settings / [Default group](../../index.mdx#slot-hierarchy) / Permission

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:wiki:default:template

**Module path：** Configuration / Wiki settings / [Default group](../../index.mdx#slot-hierarchy) / Global page templates

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-wiki-default-group-GYc0
    title: Space management
    moduleType: ones:settings:wiki:default:group
    entry: modules/ones-settings-wiki-default-group-GYc0/index.html
```

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
