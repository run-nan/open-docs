# Default group

## Requirements

|   ONES    |
| :-------: |
| v3.11.15+ |

## Overview

<Image img={require('./images/default.png')} />

We have provided slots for "Configuration" - "Pipeline settings" - "[Default group](../../index.mdx#slot-hierarchy)" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:pipeline:default:new](#onessettingspipelinedefaultnew)
- [ones:settings:pipeline:default:jenkins](#onessettingspipelinedefaultjenkins)
- [ones:settings:pipeline:default:repo](#onessettingspipelinedefaultrepo)

## Slots

#### ones:settings:pipeline:default:new

**Module path：** Configuration / Pipeline settings / [Default group](../../index.mdx#slot-hierarchy) / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:pipeline:default:jenkins

**Module path：** Configuration / Pipeline settings / [Default group](../../index.mdx#slot-hierarchy) / Link Jenkins

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:pipeline:default:repo

**Module path：** Configuration / Pipeline settings / [Default group](../../index.mdx#slot-hierarchy) / Manage repositories

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-pipeline-default-jenkins-YA0a
    title: Link Jenkins
    moduleType: ones:settings:pipeline:default:jenkins
    entry: modules/ones-settings-pipeline-default-jenkins-YA0a/index.html
```

**Supported config options：**

- [disabled](../../../../../../reference/config/plugin.yaml#disabled)
