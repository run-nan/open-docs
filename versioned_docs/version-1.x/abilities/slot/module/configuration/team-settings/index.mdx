# Team settings

## Requirements

|  ONES   |
| :-----: |
| v3.6.0+ |

## Overview

<Image img={require('./images/team-settings.png')} />

We have provided slots for "Configuration" - "Team settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:team:new](#onessettingsteamnew)
- [ones:settings:team:default](#onessettingsteamdefault)

## Slots

#### ones:settings:team:new

**Module path：** Configuration / Team settings / [new module](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:team:default

**Module path：** Configuration / Team settings / [Default group](../index.mdx#slot-hierarchy)

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-team-default-0FBl
    title: Team info
    moduleType: ones:settings:team:default
    modules:
      - id: ones-settings-team-default-V5WX
        title: Team info
        entry: >-
          modules/ones-settings-team-default-0FBl/ones-settings-team-default-V5WX/index.html
```
