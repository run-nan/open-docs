## Overview

<Image img={require('./images/test-case-plan.png')} />

We have provided slots for "ONES TestCase" - "Test plans" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:testcase:plan:header:action:new](#onestestcaseplancomponentcase)

## Slots

#### ones:testcase:plan:header:action:new

**Module path：** ONES TestCase / Test plans

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

## Usage

```yaml
modules:
  - id: ones:testcase:plan:header:action:new-2efs
    title: Test case
    moduleType: ones:testcase:plan:header:action:new
    entry: modules/ones:testcase:plan:header:action:new-2efs/index.html
```
