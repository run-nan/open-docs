|  ONES   |
| :-----: |

## Overview

<Image img={require('./images/testcase.png')} />

We have provided slots for "ONES TestCase" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:testcase:new](#onestestcasenew)
- [ones:testcase:overview](#onestestcaseoverview)
- [ones:testcase:library](#onestestcaselibrary)
- [ones:testcase:plan](#onestestcaseplan)
- [ones:testcase:report](#onestestcasereport)

## Slots

#### ones:testcase:new

**Module path：** ONES TestCase / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:testcase:overview

**ONES Requirement:** v3.11.15+

**Module path：** ONES TestCase / Overview

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:testcase:library

**Module path：** ONES TestCase / Test case library

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:testcase:plan

**Module path：** ONES TestCase / Test plans

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:testcase:report

**Module path：** ONES TestCase / Test reports

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-testcase-overview-rsuI
    title: Overview
    moduleType: ones:testcase:overview
    entry: modules/ones-testcase-overview-rsuI/index.html
```

- [disabled](../../../../reference/config/plugin.yaml#disabled)
