# Organization settings

## Requirements

| **ONES** |
| :------: |
| v3.6.0+  |

## Overview

<Image img={require('./images/organization-settings.png')} />

We have provided slots for "Organization settings" - "Organization settings" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:settings:organization:manage:new](#onessettingsorganizationmanagenew)
- [ones:settings:organization:manage:info](#onessettingsorganizationmanageinfo)
- [ones:settings:organization:manage:permission](#onessettingsorganizationmanagepermission)
- [ones:settings:organization:manage:team](#onessettingsorganizationmanageteam)
- [ones:settings:organization:manage:payment](#onessettingsorganizationmanagepayment)

## Slots

#### ones:settings:organization:manage:new

**Module path：** Organization settings / Organization settings / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:settings:organization:manage:info

**Module path：** Organization settings / Organization settings / Organization info

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:organization:manage:permission

**Module path：** Organization settings / Organization settings / Organization permission

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:organization:manage:team

**Module path：** Organization settings / Organization settings / Team management

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

#### ones:settings:organization:manage:payment

**Module path：** Organization settings / Organization settings / Authorization Management

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-settings-organization-manage-info-qdfs
    title: Organization info
    moduleType: ones:settings:organization:manage:info
    entry: modules/ones-settings-organization-manage-info-qdfs/index.html
```

- [disabled](../../../../../reference/config/plugin.yaml#disabled)
