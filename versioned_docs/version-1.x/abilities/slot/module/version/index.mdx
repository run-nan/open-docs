## Overview

<Image img={require('./images/version.png')} />

We have provided slots for "Version Management" to customize the page module for the application.

The slots currently supported are as follows:

- [ones:version:new](#onesversionnew)
- [ones:version:index](#onesversionindex)

## Slots

#### ones:version:new

**Module path：** Version Management / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

#### ones:version:index

**Module path：** Version Management / version

**Number limit：** Declare a maximum of **1** slots in each plugin, and up to **1** slots in the system.

## Usage

```yaml
modules:
  - id: ones-version-index-PJj3
    title: Version
    moduleType: ones:version:index
    entry: modules/ones-version-index-PJj3/index.html
```

**Supported config options：**

- [disabled](../../../../reference/config/plugin.yaml#disabled)
