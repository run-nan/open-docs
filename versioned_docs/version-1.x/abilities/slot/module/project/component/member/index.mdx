# Member

## Requirements

| **ONES**  |
| :-------: |
| v3.10.14+ |

## Overview

<Image img={require('./images/header-action.png')} />

We support appending a new module to the left of the top header action zone under the path "ONES Project" - "Project component" - "Member".

The slots currently supported are as follows:

- [ones:project:component:member:action:new](#onesprojectcomponentmemberactionnew)

## Slots

#### ones:project:component:member:action:new

**Module path：** ONES Project / Project component / Member / top header action / new module

**Number limit：** Declare a maximum of **10** slots in each plugin, and up to **100** slots in the system.

## Usage

```yaml
modules:
  - id: ones-project-component-member-action-new-r6pG
    title: new header action
    moduleType: ones:project:component:member:action:new
    entry: modules/ones-project-component-member-action-new-r6pG/index.html
```
