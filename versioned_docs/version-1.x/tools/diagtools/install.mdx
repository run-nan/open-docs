# Download and Installation

ONES Open Platform Diagtools download and installation.

## Command line interface Tools

### Online Installation

In the networking environment, we recommend that you use Node.js to install online. You donâ€™t need to consider the system architecture online installation, and it is easier to upgrade in the future.

Install the latest version:

```bash npm2yarn
npm install -g @ones/diagtools-cli
```

Install a specific version:

```bash npm2yarn
npm install -g @ones/diagtools-cli@[version]
```

Use:

```bash
diagtools --help
```

### Offline Installation

For the internal network environment and users who do not want to install node.js, we recommend you to download the offline compressing package, which contains all the dependencies required for the CLI tool. You can directly decompress it.

After downloading the offline package, you need to use `tar` to decompress.

```bash
tar -zxvf [name].tar.gz
```

Use:

```bash
./diagtools/bin/diagtools --help
```

Or add the decompressed `diagtools/bin/diagtools` to your system's bin directory, global use:

```bash
diagtools --help
```
